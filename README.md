# My_Jarvis_Finance_Bot——基于DeepSeek的A股自动化舆情分析监控系统

## 项目简介
本项目是一个 7x24 小时全自动运行的多智能体（Multi-Agent）金融监控系统。系统通过分布式爬虫实时捕获全球宏观快讯、A 股资金热榜及个股微观散户舆情，并利用 DeepSeek 大语言模型 (LLM) 进行深度文本解析与多空情绪量化研判，最终生成可视化 HTML 早报。

本项目旨在为量化金融研究（如：基于非结构化替代数据的情绪因子构建、市场状态依存性分析）提供高质量、带有精确时间戳的底层数据基础设施。

## 核心亮点
* **强健的底层爬虫引擎**：
  * **突破编码与反爬限制**：针对新浪股吧等老牌金融网站，处理了复杂的 GBK 中文编码解析问题，并实现了动态伪装与礼貌轮询（Politeness）机制。
  * **多维度数据融合**：整合了宏观（全网快讯）、中观（板块热度）与微观（个股散户评论）三层数据。
  * 
* **LLM 结构化数据萃取**：
  * **Prompt Engineering**：摒弃传统的文本摘要，通过提示词工程，要求LLM将海量非结构化评论转化为结构化的 `JSON` 格式（包含看多/看空/分歧信号、情绪评分及核心逻辑）。
  * 
* **工业级自动化部署**：
  * **容器化隔离**：各爬虫节点与分析节点通过 Docker 容器化独立运行，通过挂载宿主机 Volume 实现数据流转。
  * **T+1 异步调度**：引入 `schedule` 模块，巧妙处理跨日数据的时间差（T日晚间抓取舆情，T+1日清晨聚合分析）。

## 系统架构与工作流
系统由 4 个独立的 Bot 节点组成：
1. **Bot 1 (宏观抓取)**：高频轮询全球金融资讯，基于 `detection_target.txt` 关键词词库过滤高价值宏观事件。
2. **Bot 2 (热股抓取)**：定期抓取 A 股市场热度 Top 10 榜单，追踪主力资金流向。
3. **Bot 3 (股评抓取)**：每日 23:30 定时启动，根据当日热榜动态拼接股票代码（sz/sh/bj），深入股吧抓取散户实时评论，落盘为 CSV。
4. **Bot 4 ( Ai 研报)**：每日 06:00 准时唤醒，回溯聚合个股舆情与宏观事件，调用DeepSeek API进行多空研判，渲染红绿图表卡片并触发 SMTP 邮件推送。

## 技术栈
* **语言**：Python 3.10+
* **数据采集**：`requests`, `lxml` (XPath), `pandas`
* **AI 接入**：DeepSeek API
* **调度与通知**：`schedule`, `smtplib`, `email`
* **部署**：Docker
